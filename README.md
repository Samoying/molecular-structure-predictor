# molecular-structure-predictor
크루스칼 알고리즘과 최소 신장 트리를 활용한 분자 구조 예측 프로그램
# 분자 구조 생성기

이 프로젝트는 주어진 탄소(C), 산소(O), 수소(H) 원자 수에 기반하여 분자 구조를 생성하고 시각화하는 파이썬 스크립트를 제공합니다. 이는 수정된 크루스칼(Kruskal) 알고리즘과 몬테카를로(Monte Carlo) 시뮬레이션을 사용하여 단일, 이중, 삼중, 사중 결합을 포함하는 안정적인 분자 구조를 찾습니다.

## 기능

- **분자 생성**: 지정된 C, O, H 원자에 대한 분자 구조를 생성합니다.
- **원자가 기반 결합**: 화학적으로 타당한 구조를 보장하기 위해 원자 원자가(예: C=4, O=2, H=1)를 통합합니다.
- **다중 결합**: 이중, 삼중, 사중 결합을 감지하고 시각화합니다.
- **그래프 시각화**: `networkx` 및 `matplotlib`을 사용하여 생성된 분자 그래프를 시각화합니다.

## 작동 방식

1.  **입력**: 스크립트는 탄소(C), 산소(O), 수소(H) 원자의 수를 입력으로 받습니다.
2.  **초기 그래프 구성**: 모든 원자가 노드이고 잠재적 결합이 무작위 가중치를 가진 에지인 초기 그래프가 구성됩니다.
3.  **크루스칼 알고리즘 (몬테카를로 1)**: 몬테카를로 시뮬레이션(`monte_carlo` 메서드) 내에서 수정된 크루스칼 알고리즘(`kruskal` 메서드)을 사용하여 원자 원자가를 존중하는 초기 신장 트리를 찾습니다. 이 단계는 주로 가능한 한 많은 원자가를 만족시키고 연결된 구조를 생성하기 위해 단일 결합을 형성하는 데 중점을 둡니다.
4.  **고리 및 다중 결합 형성 (몬테카를로 2 & 3)**: 초기 신장 트리 후에 원자가가 남아 있으면 `monte_carlo2` 메서드는 추가 결합(이중, 삼중 또는 사중) 및 고리를 형성하려고 시도합니다. `monte_carlo3` 메서드는 나머지 원자가를 만족시키기 위해 다중 결합 형성을 특별히 처리합니다.
5.  **시각화**: 다중 결합을 포함한 최종 분자 구조는 `networkx` 및 `matplotlib`을 사용하여 시각화됩니다.

## 사용법

스크립트를 사용하려면 `C`, `O`, `H` 변수에 탄소, 산소, 수소 원자의 수를 설정해야 합니다. 그런 다음 코드 셀을 순서대로 실행합니다.

### 예시:

```python
C = 6
O = 0
H = 6

# 노트북의 나머지 코드는 분자를 생성하고 시각화합니다.
# 예를 들어, C6H6 (벤젠) 구조를 생성합니다.
```

## 코드 구조

-   **`Graph` 클래스**: 그래프 작업(에지 추가, 크루스칼을 위한 집합 찾기/병합, 몬테카를로 알고리즘 포함)을 캡슐화합니다.
    -   `__init__(self, v)`: `v`개의 정점, 에지 목록, 크루스칼을 위한 부모 배열 및 원자가 목록으로 그래프를 초기화합니다.
    -   `add(self, a, b, w)`: `a` 노드와 `b` 노드 사이에 가중치 `w`를 가진 에지를 추가합니다.
    -   `find(self, v)`: `v`를 포함하는 집합의 대표(루트)를 찾습니다.
    -   `union(self, a, b)`: `a`와 `b`를 포함하는 집합을 병합합니다.
    -   `kruskal(self)`: 원자가를 존중하면서 최소 신장 트리를 구축하기 위한 수정된 크루스칼 알고리즘을 구현합니다.
    -   `monte_carlo(self, C, O, H)`: 초기 단일 결합 구조를 구축하는 첫 번째 몬테카를로 단계입니다.
    -   `monte_carlo2(self, C, O, H, initial_valence)`: 이중/삼중/사중 결합 및 고리 형성을 처리합니다.
    -   `monte_carlo3(self)`: 모든 원자가가 만족되거나 더 이상 결합을 형성할 수 없을 때까지 다중 결합을 재귀적으로 형성합니다.

-   **주요 실행 블록**: 원자의 수를 설정하고, `Graph` 클래스 메서드를 호출하고, 다중 결합을 식별하기 위해 결과를 처리한 다음, 최종 분자 구조를 시각화합니다.
